# PetCare 宠护 - 产品需求文档

## 项目概述

**项目名称**: PetCare 宠护  
**版本**: v1.0 MVP  
**创建时间**: 2026-03-23  
**目标**: 帮助宠物主人科学管理宠物健康，不再忘记疫苗驱虫

---

## 用户画像

### 主要用户
- **年龄**: 20-40 岁
- **城市**: 一二线城市
- **特征**: 养宠人群，尤其是多宠物家庭
- **痛点**: 
  - 忘记疫苗/驱虫时间
  - 多宠物管理混乱
  - 缺乏科学养宠知识

---

## 核心功能 (MVP)

### 1. 宠物档案管理
- [ ] 创建宠物档案（名字、品种、生日、性别）
- [ ] 上传宠物照片
- [ ] 多宠物管理（支持无限数量）
- [ ] 宠物信息编辑/删除

### 2. 疫苗提醒
- [ ] 疫苗记录录入（手动/扫码）
- [ ] 自动计算下次接种时间
- [ ] 微信/短信/推送提醒
- [ ] 疫苗记录查看/编辑
- [ ] 疫苗本导出（PDF）

### 3. 驱虫提醒
- [ ] 驱虫记录录入
- [ ] 体内外驱虫分类
- [ ] 自动计算下次驱虫时间
- [ ] 推送提醒

### 4. 知识库
- [ ] 疫苗知识科普
- [ ] 驱虫知识科普
- [ ] 分品种养护指南
- [ ] 饮食禁忌查询

---

## 技术栈

### 前端
- **框架**: UniApp (Vue3)
- **UI 库**: uView UI
- **目标平台**: 微信小程序 + H5

### 后端
- **语言**: Node.js 18+
- **框架**: Express / Fastify
- **数据库**: MySQL 8.0
- **缓存**: Redis
- **部署**: Docker + 腾讯云

### 第三方服务
- **推送**: 微信小程序模板消息
- **存储**: 腾讯云 COS
- **短信**: 腾讯云 SMS

## 成功指标

### 产品指标
- 次日留存 > 50%
- 7 日留存 > 35%
- 30 日留存 > 25%

### 用户指标
- 种子用户 1000 人
- 付费转化率 > 5%
- NPS > 50

---

## 风险与应对

| 风险 | 应对 |
|------|------|
| 推送不及时 | 多通道推送 |
| 用户增长慢 | 多渠道测试 |
| 付费意愿低 | 免费 + 增值模式 |
