## ⚠️ 常见审核问题

### 1. 功能不完整
**问题**: 提交的版本功能缺失  
**解决**: 确保所有展示的功能都可用

### 2. 隐私政策缺失
**问题**: 未提供隐私政策  
**解决**: 在小程序内添加隐私政策页面

### 3. 类目选择错误
**问题**: 选择了需要资质的类目  
**解决**: 选择工具类，避免医疗相关描述

### 4. 测试账号问题
**问题**: 需要登录但没提供测试账号  
**解决**: 在备注中提供测试账号或说明无需登录

### 5. 名称违规
**问题**: 名称包含敏感词  
**解决**: 使用中性名称，避免夸大宣传

## 📤 提交流程

### 步骤 1: 代码上传
```bash
# 使用 HBuilderX 或 uni-app CLI
npm run build:mp-weixin

# 上传到微信开发者工具
# 文件 → 上传
```

### 步骤 2: 提交审核
1. 登录 https://mp.weixin.qq.com
2. 版本管理 → 开发版本
3. 选择刚上传的版本
4. 点击 **提交审核**

### 步骤 3: 填写信息
1. 填写功能页面
2. 选择服务类目
3. 上传截图
4. 填写测试账号 (如需要)
5. 确认隐私政策

### 步骤 4: 等待审核
- **审核时间**: 1-7 个工作日
- **审核结果**: 短信 + 站内信通知

### 步骤 5: 发布上线
1. 审核通过后，登录后台
2. 点击 **发布**
3. 用户即可搜索到小程序

## 🎯 审核技巧

### 提高通过率

1. **功能完整**: 确保所有功能可用
2. **内容健康**: 无违规内容
3. **体验良好**: 无崩溃、无卡顿
4. **隐私合规**: 完整的隐私政策
5. **类目正确**: 选择正确的服务类目

### 避免被拒

1. ❌ 不要涉及医疗诊断
2. ❌ 不要使用绝对化用语
3. ❌ 不要收集无关信息
4. ❌ 不要有未实现功能
5. ❌ 不要侵犯他人权益

## 📞 审核被拒处理

### 如果被拒绝

1. **查看拒绝原因**
   - 登录后台查看审核意见
   - 仔细阅读拒绝原因

2. **修改问题**
   - 根据审核意见修改
   - 确保问题完全解决

3. **重新提交**
   - 修改后重新上传代码
   - 再次提交审核
   - 在备注中说明修改内容

4. **申诉**
   - 如认为审核有误
   - 可提交申诉
   - 提供相关证明材料
