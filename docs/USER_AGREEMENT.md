## 一、协议说明

### 1.1 协议主体

本用户协议（以下简称"本协议"）由您（以下简称"用户"）与 PetCare 宠护运营方（以下简称"我们"）共同缔结。

### 1.2 协议范围

本协议适用于 PetCare 宠护应用（包括但不限于微信小程序、iOS App、Android App、鸿蒙 App、H5 等所有版本）。

### 1.3 协议生效

**当您注册、登录或使用 PetCare 时，即表示您已阅读、理解并同意接受本协议的全部内容。**

## 二、服务内容

### 2.1 核心功能

PetCare 为您提供以下服务：

1. **宠物档案管理**: 创建和管理宠物信息
2. **健康记录**: 记录疫苗、驱虫、健康数据
3. **智能提醒**: 疫苗/驱虫到期提醒
4. **知识库**: 科学养宠知识查询
5. **其他功能**: 我们可能不时推出的新功能

### 2.2 服务变更

我们保留以下权利：

- 增加或减少功能
- 暂停或终止部分服务
- 调整服务收费（如有）

我们会提前通知重大变更。

## 三、账号管理

### 3.1 账号注册

您可通过以下方式注册：

- 微信一键登录
- 手机号 + 验证码

**您应保证注册信息的真实性、准确性。**

### 3.2 账号安全

您应：

- 妥善保管账号信息
- 不将账号出借、转让给他人
- 如发现账号被盗，立即通知我们

### 3.3 账号注销

您可随时注销账号：

- 路径：个人中心 → 设置 → 注销账号
- 注销后，我们将删除您的所有信息
- 注销后，账号不可恢复

## 四、用户行为规范

### 4.1 允许的行为

您可以：

- 使用 PetCare 提供的各项功能
- 管理自己的宠物档案
- 记录宠物健康数据

### 4.2 禁止的行为

您不得：

1. **违法内容**: 发布违法违规信息
2. **侵权内容**: 侵犯他人知识产权、肖像权等
3. **虚假信息**: 发布虚假宠物信息
4. **恶意行为**: 攻击、侵入系统
5. **商业行为**: 未经同意进行商业推广
6. **其他**: 其他违反法律法规的行为

### 4.3 违规处理

如我们发现您有违规行为，我们有权：

- 删除违规内容
- 暂停或终止服务
- 向有关部门报告
- 追究法律责任

### 7.1 服务中断

PetCare 可能包含第三方链接或服务：
