# PetCare UI 设计文档

## 设计规范

### 色彩体系

**主色调**:
- 🟠 橙色：`#FF9500` (温暖、活力)
- ⚪ 白色：`#FFFFFF` (背景)
- ⚫ 深灰：`#333333` (文字)
- 🔵 蓝色：`#007AFF` (链接/操作)
- 🟢 绿色：`#34C759` (成功/健康)
- 🔴 红色：`#FF3B30` (警告/过期)

### 字体规范

- **标题**: PingFang SC Bold, 18-24px
- **正文**: PingFang SC Regular, 14-16px
- **辅助文字**: PingFang SC Light, 12px

### 圆角规范

- **卡片**: 12px
- **按钮**: 8px
- **头像**: 50% (圆形)

## 交互流程

### 1. 创建宠物流程
```
首页 → 点击"+" → 添加宠物页 → 填写信息 → 上传照片 → 保存 → 返回首页
```

### 2. 添加疫苗记录流程
```
首页 → 选择宠物 → 宠物详情 → 记录疫苗 → 填写信息 → 上传照片 → 保存 → 设置提醒
```

### 3. 查看提醒流程
```
首页 → 待办提醒卡片 → 详情 → 标记已完成 / 推迟
```
