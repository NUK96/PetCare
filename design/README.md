## 🔄 交互流程

### 创建宠物
```
首页 → 添加 → 填写信息 → 上传照片 → 保存 → 首页显示
```

### 添加疫苗
```
宠物详情 → 记录疫苗 → 选择疫苗 → 填写日期 → 上传照片 → 保存 → 设置提醒
```

### 查看提醒
```
首页 → 提醒卡片 → 查看详情 → 标记完成/推迟
```
