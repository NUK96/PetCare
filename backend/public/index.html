🐾 PetCare 宠护

科学养宠，从记录开始

服务状态：✅ 运行中

端口：3000

环境：development

数据库：SQLite

📋 主要 API 接口

  • GET /api/v1/health
    健康检查
  • GET /api/v1/knowledge
    获取知识库列表
  • POST /api/v1/auth/login
    微信登录
  • GET /api/v1/pets
    获取宠物列表 (需登录)
  • GET /api/v1/vaccines
    获取疫苗记录 (需登录)
  • GET /api/v1/dewormings
    获取驱虫记录 (需登录)
